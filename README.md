# Survival Game

A simple browser-based survival game built with HTML, CSS, and JavaScript.

## How to play

- Open `index.html` in a web browser.
- Use arrow keys or WASD to move.
- Collect green supplies to restore health.
- Avoid red enemies as they drain your health over time.
- Survive as long as possible.

## Files

- `index.html` — game structure and UI.
- `style.css` — game styling.
- `script.js` — gameplay logic.
